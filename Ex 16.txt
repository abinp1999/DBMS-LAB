2.Create a trigger to handle inserts to the product relation ( if price < 5 the set price to 10).

mysql> create trigger inserttrigger
    ->         before insert 
    ->         on product for each row
    ->            begin
    ->                if new.price < 5 then
    ->                     set new.price=10;
    ->                 end if;
